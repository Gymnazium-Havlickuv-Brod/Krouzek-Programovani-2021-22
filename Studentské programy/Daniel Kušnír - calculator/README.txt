Vlastnoručně napsaná kalkulačka, která by šla nahradit --> result =  eval("příklad")

Matematické operace:
-----------------------------------------------------------------
|    mínus = - (9-6)							                |
|    plus = + (9+6)							                    |
|    krát = * (9*6)							                    |
|    děleno = / (9/6)						                    |
|    mocnina = ** (9**6) --> devět na šestou				    |
|    odmocnina = // (27//3) --> třetí odmocnina z 27			|
-----------------------------------------------------------------

Pokud chcete napsat zápornou hodnotu, tak ji vždy dejte na začátek závorky nebo na začátek příkladu:
-------------------------
|	-9+6		        |
|	(-9)+6		        |
|	6+(-9)		        |
|	(-9+6)		        |
|	(6+(-9))	        |
-------------------------
!!! Pro počítání desetinných čísel použij čárku (,) nebo tečku (.) (Př. 5,32 nebo 5.32)
!!! Neumí to počítat neznámé nebo-li písmenka (a,b,c,d,...)
!!! Takový těžší příklad na vyzkoušení: 
-------------------------------------------------------------------------------------------------
|	((-3**9+65*(35*(-2)+3)/((-6*9)*(-3*7))+652)/2034//2)**2/35-(9*5)-7**2+(4**3-16)		        |
-------------------------------------------------------------------------------------------------
